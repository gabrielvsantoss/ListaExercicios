# Lista de Exercicios

## Funcionalidades Explicadas por exercicio:

- **Exercicio 01:** Algoritmo que calcula o volume de uma caixa retangular.

- **Exercicio 02:** Algoritmo que calcula o volume de um cilindro.

- **Exercicio 03:** Algoritmo que calcula a quantidade conbustivel gasto por km de uma determinada distancia digitada pelo usuario.

- **Exercicio 04:** Algoritmo transforma um temperatura em gruas Celsius digitada pelo usuario em Fahrenheit.

- **Exercicio 05:** Algoritmo que calcula uma comissão para um funcionario com base em suas vendas ( a comissão em porcentagem é digitada pelo usuario).

- **Exercicio 06:** Algoritmo que calcula  a media harmonica de tres notas digitadas pelo usuario.

- **Exercicio 07:** Algoritmo que calcula a media ponderada de duas notas digitadas pelo usuario.
 
- **Exercicio 08:** Algoritmo que verifica se um numero digitado pelo usuario é primo.

- **Exercicio 09:** Algoritmo que calcula a area de um terreno retangular.
 
- **Exercicio 10:** Algoritmo que calcula quantos pães e quantas broas uma padaria vendeu e retorna o faturamento das vendas.
 
- **Exercicio 11:** Algoritmo que calcula quantos dias uma pesssoa ja viveu.

- **Exercicio 12:** Algoritmo que adiciona uma comissão de 15% para um funcionario e depois retira 8% de comissão.
 
- **Exercicio 13:** Algoritmo que recebe tres valores (A, B e C) e depois imprime na tela se a soma de A + B é menor que C.
 
- **Exercicio 14:** Algoritmo que recebe tres valores e imprime eles de forma decrescente
 
- **Exercicio 15:** Algoritmo que recebe a altura e o peso de uma pessoa e calcula o IMC dela com base em alguns criterio.
 
- **Exercicio 16:** Algoritmo que recebe um valor ee verifica se ele é par ou impar.
 
- **Exercicio 17:** Algortimo que le dois valores inteiros (A e B) e depois verifica se os numeros são iguais se eles forem os numeros são somados se forem diferentes os numeros são multiplicados.
 
- **Exercicio 18:** Algoritmo que imprime na tela todos os numeros impares entre 100 e 200.
 
- **Exercicio 19:** Algoritmo que verifica todos os numeros entre 1 e 500 que são impares e são multiplos de 3, soma eles e retorna o valor.
 
- **Exercicio 20:** Algoritmo que recebe um numero e retorna a tabuada do mesmo.

- **Exercicio 21:** Algoritmo que calcula o fatorial de um valor inteiro digitado pelo usuario.

- **Exercicio 22:** Algoritmo que le um numero e imprime a sequencia de Fibonacci ate esse numero.

- **Exercicio 23:** Algoritmo que verifica se os numeros de 1 a 100 e imprimi "Fizz" se o numero for multiplo de 3, "Buzz" se o numero foi multiplo de 5, "FizzBuzz" se o numero for multiplo de 5 e 3, retorna o proprio numero se o numero não estiver em nenhuma dessas condições

**Como usar?**

1. Clone o repositorio ou baixe o código fonte.
2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz.
3. Utilize o comando abaixo para restaurar as dependencias do projeto:

```
dotnet restore
```

4. Em seguida, complete uma solução utilizando o comando:
```
dotnet build --configuration Release 
```

5. Para executar o projeto elaborado em tempo real:
```
dotnet run --project ListaExercicios
````

6. Para executar o arquivo compilado, navegue até a pasta: ./ListaExercicios/bin/Release/net8.0/e execute o arquivo:
```
ListaExercicios.exe
```

**Requisitos**

- NET SDK (recomendado .NET 8.0 ou superior) para construção e execução do projeto.
